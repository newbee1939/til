## Linux プロセス: 経過時間と使用時間

- `経過時間`
    - プロセスが開始してから終了するまでの経過時間
- `使用時間`
    - プロセスが実際に論理CPUを使用した時間

## Go: selectブロック

- 複数のチャネル操作を待ち受け、準備ができたものを1つ実行する構文
    - `switch` のチャネル版のイメージ
- 挙動
    - 実行可能な `case` が1つだけ → それを実行
    - 複数ある → ランダムに1つ選ばれる（偏りを防ぐため）
    - 1つもない → `default` があればそれを実行、なければブロックして待つ

```go
select {
case v := <-ch1:
    // ch1 から受信できたとき
    fmt.Println(v)
case ch2 <- x:
    // ch2 へ送信できたとき
case <-time.After(time.Second):
    // 1秒間どのcaseも準備できなかったとき（タイムアウト）
default:
    // どのcaseもすぐ実行できないとき（非ブロッキング）
}
```
