## n8n: manual installation

https://docs.n8n.io/integrations/community-nodes/installation-and-management/manual-installation

コミュニティノード（第三者が npm に公開している `n8n-nodes-*` パッケージ）を、GUI ではなく npm で直接入れる方法。

### 手順

インストール先は `~/.n8n/nodes`。ここに置いた npm パッケージを n8n が**起動時に**読み込む。

```sh
docker exec -it n8n sh
mkdir ~/.n8n/nodes && cd ~/.n8n/nodes
npm i n8n-nodes-nodeName
```

最後に n8n を再起動する。あとは `npm uninstall` / `npm update` と普通の npm 操作そのまま。バージョン指定で戻したいときだけ、uninstall してから `npm i n8n-nodes-nodeName@2.1.0`。
