## ⭐️AWS DevOps Agent

https://aws.amazon.com/jp/blogs/news/announcing-general-availability-of-aws-devops-agent/

- AWSがDevOps AgentをGA（一般提供開始）。「常時待機する運用チームメイト」として位置づけ
- インシデント検知 → 調査 → 復旧 → 予防 のライフサイクル全体を自律的にこなすAIエージェント
- プレビュー期間中の実績: **MTTR**（Mean Time To Repair = 障害発生から正常復旧までの平均時間）最大75%削減・調査時間80%短縮・根本原因特定精度94%
- Datadog / Dynatrace / New Relic / Splunk / PagerDuty / Grafana / GitHub / GitLab など主要ツールと統合済み
- GAで Azure・Azure DevOps・オンプレミス（MCP経由）にも対応し、マルチクラウドをカバー
- 東京を含む6リージョンで利用可能。料金は秒単位の従量課金

## wolfGuard

https://github.com/wolfssl/wolfguard

- **wolfSSL** とは: IoT・組み込み機器向けに特化した軽量TLS/暗号ライブラリ。フットプリント（ソフトウェアやハードウェアが動作するために必要なリソース量）が小さく、メモリが数十KB単位の機器でもTLS通信を実現できる。組み込みセキュリティ分野での実績が豊富
- wolfSSLが新たに開発する **wolfGuard** は、その暗号・認証の知見を活かした「デバイス保護」ライブラリ。セキュアブート・鍵管理・改ざん検知といった、TLS通信の前後を守る機能群と見られる

## ⭐️Shell Tricks That Actually Make Life Easier (And Save Your Sanity)

https://blog.hofstede.it/shell-tricks-that-actually-make-life-easier-and-save-your-sanity/

- **カーソル操作（ほぼ全シェル共通）**
  - `Ctrl+W` : カーソル前の1ワードを削除
  - `Ctrl+U` : カーソルから行頭/行末まで切り取り（`Ctrl+Y` で貼り付け）
  - `Ctrl+A` / `Ctrl+E` : 行頭/行末へ瞬時移動
- **地味に便利な系**
  - `cd -` : 直前のディレクトリへトグル移動
- **Bash/Zsh限定**
  - `Ctrl+R` : コマンド履歴のインクリメンタル検索
  - `sudo !!` : 直前コマンドをsudoで再実行（`Permission denied` の直後に打つやつ）
