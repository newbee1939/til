## git -C

- cd コマンドで移動しなくても、指定した別のディレクトリにいる体で git コマンドを実行できる
- 実際のカレントディレクトリは移動しない

```shell
# -Cを使わない場合
cd /path/to/project
git status

# -Cを使った場合
git -C /path/to/project status
```

## ⭐️navigator.webdriver = true

### 1. 概要

`navigator.webdriver = true` は、ブラウザがプログラムによって自動操縦されている状態を示すフラグ

### 2. 仕組み

- ブラウザの属性: JavaScriptの `navigator` オブジェクトに存在する項目
- 手動アクセス: 人間が操作する場合は false または undefined
- 自動アクセス: Selenium等の自動化ツール起動時、ブラウザが自動で true に設定
- 隠せない足跡: システム内部で強制的に書き換わるため、通常の起動では隠蔽困難

### 3. Webサイト側の用途

- Bot検知: JavaScriptでこのフラグを監視
- アクセス制御: true（Bot判定）の場合にブロックや画像認証を要求

> ブラウザ自らが差し出す「私はロボットです」という名札
