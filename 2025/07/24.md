## Gemini CLIの料金について

参考: https://blog.g-gen.co.jp/entry/gemini-cli-explained

- 基本的には、Gemini CLI は無料で利用できる
- より多くのリクエストが必要な場合、使用量ベース課金の Google AI Studio または Vertex AI、あるいは有償版の Gemini Code Assist Standard / Enterprise ライセンスを利用することも可能
- また Gemini Code Assist Standard / Enterprise ライセンスで Gemini CLI を利用すると、利用量の制限が緩和されるほか、Gemini Code Assist と連携して VS Code で Gemini Code Assist エージェントモード が利用可能になる

## ESNI（Encrypted Server Name Indication）

https://www.cloudflare.com/ja-jp/learning/ssl/what-is-sni/

- SNIはドメイン名を`平文で`送信し、複数のSSL証明書を管理するための技術
- 暗号化された`SNI（ESNI）`は、Client HelloのSNI部分を暗号化することで、SNI拡張機能に追加する
- これにより、クライアントとサーバーの間をのぞき見する者は、クライアントがどの証明書を要求しているかを知ることができなくなり、クライアントの保護とセキュリティがさらに強化される
- CloudflareとMozilla Firefoxは、2018年にESNIのサポートを開始した
- ただし、クライアントとサーバーの両方がESNIをサポートしている必要がある

## Akamai Property: Cache Key Hostname

- Origin Hostname (オリジンホスト名)
  - 概要: オリジンサーバーのホスト名をキャッシュの基準にします。
  - 用途: どのドメインでアクセスされても、常に同じコンテンツを返す場合。
- Incoming Hostname (リクエストホスト名)
  - 概要: ユーザーがアクセスした際のホスト名をキャッシュの基準にします。
  - 用途: アクセスされたドメインによって表示するコンテンツを変えたい場合（例：仮想サーバー）。

## Google Cloud: serverIp, remoteIp

https://cloud.google.com/logging/docs/reference/v2/rest/v2/LogEntry#httprequest

- remoteIp
  - `The IP address (IPv4 or IPv6) of the client that issued the HTTP request`
- serverIp
  - `The IP address (IPv4 or IPv6) of the origin server that the request was sent to`

アクセス元のIPを確認したいときは、`remoteIp`を参照する。
