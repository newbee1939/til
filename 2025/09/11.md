## nginx: proxy_ssl_server_name

参考: https://nginx.org/en/docs/http/ngx_http_proxy_module.html#proxy_ssl_server_name

- このディレクティブが`on`に設定されている場合、nginxはクライアントがリクエストしたホスト名を`SNI`としてプロキシ先サーバーに送信する

## AGENTS.md の概要

https://note.com/npaka/n/nd1258df2853c

- 「AGENTS.md」は、コーディングエージェント向けの「README」のようなもの
- 意図的に「AGENTS.md」分離したのは、以下の理由から
  - エージェントが指示を記述できる、明確で予測可能な場所を提供する
  - README は簡潔で、人間の貢献者に焦点を当てたものにする
  - 既存の README やドキュメントを補完する、エージェント向けの正確なガイダンスを提供する
