## Claude Code Actions

参考: https://zenn.dev/acntechjp/articles/3f361da473eac8

- 概要
  - Claude Code ActionはGitHubでPull RequestやIssue向けに、Claude Codeに質問への回答やコード変更の実装を行わせるアクション
  - AnthropicダイレクトAPI(従量課金)に対応
- 活用例
  - `@claude`メンションでPRやIssueから直接呼び出し
  - 自動PR作成 - Issueから実装を生成
  - コードレビュー - PRの自動レビュー
  - バグ修正 - 問題を検出して修正PRを作成

従量課金制なのが若干不安な気はする...

## Linux: カーネルモジュール

- カーネルを拡張するもの
- 一般的に`/lib/modules/<カーネルバージョン>/kernel`に存在
- 基本的には起動時に自動的にロードされる
- ロードされているモジュールは`lsmod`コマンドで確認できる
