## Gemini for Google Cloud

- 「Gemini for Google Cloud」は、同社の最新AIモデルである「Gemini」を用いた複数のサービスの統合的なブランド
- コーディング支援のGemini Code AssistもGemini for Google Cloudの傘下と位置づけられている

- https://www.publickey1.jp/blog/24/gemini_for_google_cloudgoogle_cloudaigoogle_cloud_next_24.html
- https://cloud.google.com/blog/products/ai-machine-learning/gemini-for-google-cloud-is-here?hl=en

`Gemini for Google Cloud`の料金については、以下の資料に書かれている。

https://cloud.google.com/products/gemini/pricing?hl=ja

Gemini Cloud Assistの料金は以下に書かれている。

https://cloud.google.com/products/gemini/cloud-assist?hl=ja#pricing

Gemini Cloud Assist in Cloud Billing

※Cloud BillingのGemini Cloud Assistは追加料金なしで利用できる。Gemini for Google Cloudを他のGoogle Cloud製品（BigQuery、Gemini Code Assist、またはLookerなど）と一緒に使用すると、料金が発生したり、追加または別の機能の購入が必要になる場合がある

https://cloud.google.com/billing/docs/how-to/gemini/overview
